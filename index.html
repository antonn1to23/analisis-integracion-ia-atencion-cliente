1. Caso de trabajo

Una empresa mediana de servicios B2B quiere mejorar su atención al cliente usando IA generativa para resumir tickets, sugerir respuestas, buscar información en documentos internos y reducir el tiempo medio de respuesta. La revisión humana es obligatoria y la trazabilidad es alta.

Sistemas y condiciones del caso
  • CRM, ERP, Portal de tickets y Base documental interna.
  • Equipo humano de soporte de 35 personas.
  • Datos personales (PII) dentro de algunos tickets.
  • Revisión humana obligatoria antes de enviar respuestas al cliente.
  • Latencia máxima deseada: 4 segundos por respuesta.
Alternativas a comparar
  • Opción A — API externa de IA: conexión por API con un proveedor externo. Rapidez de implantación frente a coste variable y dependencia.
  • Opción B — Plataforma cloud gestionada: servicio de IA dentro del proveedor cloud principal. Mejor trazabilidad y permisos frente a complejidad inicial.
